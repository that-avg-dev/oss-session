# OSS Session 🚀

Welcome to **OSS Session**, a safe and welcoming space for students to make their first Open Source contributions!

## Why Contribute? 🌟

Contributing to Open Source Software (OSS) is a fantastic way to learn new skills, collaborate with others, and build a portfolio of work. Here, you'll learn:

- How to fork a repository
- How to create a branch
- How to commit changes
- How to open a Pull Request (PR)
- How to pass automated PR checks

## Your Reward! 🏆

Every student who successfully merges their first Pull Request will receive the exclusive **Awesome OSS Contributor Badge**! 🥇 Add it to your GitHub profile or resume to show off your new skills.

## How to Get Started

1. Check out the **Issues** tab. We've created several beginner-friendly issues that require NO coding experience!
2. Pick an issue and leave a comment saying "I would like to work on this!"
3. Fork this repository and clone it to your local machine.
4. Make your changes in a new branch.
5. Push your branch and open a Pull Request!

## PR Checks ✅

We have automated checks that run on every Pull Request. These checks are common in real-world OSS projects to ensure code quality. Our checks simply verify that your Markdown files are formatted correctly. Don't worry if they fail at first—you can always update your PR to fix them!
